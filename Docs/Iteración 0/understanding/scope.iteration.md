# Contexto y objetivo de Iteración 0

## Nota

El objetivo de la iteración 0 es dar el escenario actual en donde se desenvolverá la iteración 1.

No se busca diseñar toda la solución. El objetivo solo dar el pie inicial para saber como podemos faciltiar el proceso de compra del hogar, sin necesidad de implicar grandes esfuerzos.

## Contexto

La necesidad observada es que la lista mensual de compra debe estar preparada antes del pago, ya que uno de los problemas que suelen tener las familias joeves es omitir pagos importantes por presiones externas.

Si la lista no existe en momento del pago, la compra puede posponerse  y el dinero destinado a despensa puede dispersarse antes de cubrir los productos esenciales.

La lista debe quedar preparada de una forma que reduzca carga ejecutiva y permita actuar rápidamente cuando el dinero esté disponible.

## Por qué importa

Estos contexto es importante para poder resolver este problema de forma eficiente y efectivamente, ya que delimitando el que hace, sabemos que investigar y que no.
